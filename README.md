# AI Infrastructure 6-12 Month Portfolio Lab

Static GitHub Pages dashboard for tactical AI infrastructure portfolio models.

## Contents

- `index.html` - self-contained interactive dashboard.
- `build_ai_infra_dashboard.mjs` - generator used to refresh quotes and rebuild the dashboard.
- `.nojekyll` - disables Jekyll processing on GitHub Pages.

## Notes

The dashboard is educational research, not personalized financial advice. It uses quote data captured at generation time and should be verified with a broker before trading.
